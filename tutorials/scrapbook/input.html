Creating digital scrapbooks with neftox

11. April 2024


Scrapbooks work very similar to talks, so it is recommended to look at the talk tutorial where all the basics of neftox are explained.


Content creation happens within templates and boxes, just like for presentation slides. You can view the different templates at the end of the tutorial.




You can add text and images and style them in any way you want using HTML and CSS. LaTeX expressions are possible too (see talk tutorial).


--BOX1 BOX2--

The tool comes with a variety of style elements you can add to your scrapbook, like plain and wavy borders, masking tape strips, paper clips, stamps, artsy backgrounds, ... which are located in the tool's styles/images/ directory. Have a look at the input code of this tutorial on how to use them!
